Bauhaus Modern — display typeface used for the chromonym demo wordmark.

  Designer: Nils Kähler (2016)
  Source:   https://www.dafont.com/bauhaus-modern.font
  License:  "Free for personal use" (dafont listing)

Embedding on this public OSS demo site is done with the designer's
knowledge — a donation was made and the use case (MIT-licensed library
demo, display-type only, with visible attribution) was described to
the author via email. If the designer objects, remove this font and
the @font-face block referencing it from demo/src/index.css.

Please do not repackage or redistribute the font file outside the
context of this demo without following up with the designer directly.
