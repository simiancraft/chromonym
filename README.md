# chromonym

Color naming tool — resolve RGB or Hex color input to a human-readable name.

> Scaffold stage — public API is not yet defined.

## Install

```sh
bun add chromonym
# or
npm install chromonym
```

## Development

```sh
bun install
bun run lint       # biome
bun run typecheck  # tsgo (native-preview)
bun test
bun run build
```
